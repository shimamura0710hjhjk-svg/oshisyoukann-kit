# oshisyoukann-kit
推し召喚キット。LLMのAPI叩いてTTSに流してHTMLで見れるようにするあれです。ローカルホストでつないで表示すれば、表示はスマホ裏の動きはPCでできるようになっています。推し君召喚したいんだよおおおお！さあキミもこれでイチャイチャしよう！

🎭 推し召喚・パーソナルダッシュボード v1.0
このプロジェクトは、**「理想の推しを、理想の声で、手のひら（スマホ）の中に」**をコンセプトにした、個人用AIキャラクター管理システムだにゃ！

🌟 このシステムでできること
深すぎるキャラ設定: JSON形式で、MBTI、職業、独占欲の強さまでミリ単位で調整可能。

スマホ完結型管理: 外出先や布団の中から、推しの性格・画像・ボイスをリアルタイム更新。

3段階の感情ボイス: 「通常」「囁き」「圧」の3つの感情ごとに参照音声をスマホから即座に割り当て。

爆安・高音質: Gemini API (1.5 Flash) × GPT-SoVITS により、ほぼ無料〜ジュース代程度の維持費で最高品質の体験を。

🛠️ 準備するもの
Python 3.10以上 (Dドライブ等へのインストールを推奨)

Google AI Studio API Key (Gemini用)

GPT-SoVITS (ローカルでAPIサーバーとして起動)

🚀 セットアップ手順
1. ライブラリのインストール
ターミナルで以下の呪文を唱えてにゃ：

Bash
pip install -r requirements.txt
2. 環境変数の設定
フォルダ内に .env ファイルを作成し、あなたのAPIキーを書き込んでにゃ：

Plaintext
GEMINI_API_KEY=ここにあなたのAPIキー
3. 起動の二刀流
このシステムは「脳（Gemini）」と「声（SoVITS）」の2つのサーバーを同時に動かすにゃ。

声の準備: GPT-SoVITSのフォルダでAPIサーバーを起動。

Bash
# 9880ポートで待機していることを確認
python api.py
体の起動: 本プロジェクトのフォルダでダッシュボードを起動。

Bash
streamlit run app.py
📱 スマホから接続する方法
PCで streamlit run を実行した際に出る Network URL: http://192.168.x.x:8501 にスマホのブラウザでアクセスしてにゃ！

Tips: 同じWi-Fiに繋いで、スマホのホーム画面に「ショートカット」を作っておくと、次から1タップで彼に会えるよ。

📁 フォルダ構造の秘密
Cドライブを汚さないための「隔離設計」だにゃ：

/characters: 推し君たちの性格設定（JSON）。

/images: 推しのアイコン画像。背景透過PNGがおすすめ。

/voice_samples: スマホから上げたボイスたち。

/users: あなた（夢主）の設定。
